Navigate with top buttons.
Drop images.jpg/png/gif (you might have to drop it twice)
Numbers will be calculated.
This will calculate 63 bytes of data.
This numbers can be used in any commodore 64 program.

Navigieren Sie mit den oberen Tasten.
Bilder ablegen.jpg/png/gif (möglicherweise müssen Sie es zweimal ablegen)
Zahlen werden berechnet.
Dadurch werden 63 Byte Daten berechnet.
Diese Nummern können in jedem Commodore 64-Programm verwendet werden.
